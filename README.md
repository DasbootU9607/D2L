# D2L Learning Notes

This repository contains my personal notes, code practice, and experiment records while studying *Dive into Deep Learning* (D2L) and related deep learning materials.

The main goal is to document my learning process, reproduce key examples from the course, and build a clear reference that I can revisit later.

## Contents

- `notes/`: chapter notes, summaries, and key takeaways
- `code/`: code implementations and practice exercises
- `experiments/`: small experiments, model tests, and result comparisons
- `d2l/`: downloaded D2L materials or reference notebooks

## Learning Focus

- Deep learning fundamentals
- PyTorch basics and model implementation
- Linear neural networks
- Multilayer perceptrons
- Convolutional neural networks
- Recurrent neural networks
- Optimization and training techniques

## Status

This repository is a work in progress. I will keep adding notes, code, and reflections as I move through the course.
